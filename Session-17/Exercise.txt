1. Για να κάνουμε ένα request προς ενα API χρειάζεται να προσθέσουμε το url του API και όχι το δικό μας
2.Το content type θα είναι image/jpg
3.Θα πρέπει να προσθέσουμε ένα header με max-age=86.400
4.Λογικά ναι αλλά θα μπορεί μόνο να παίρνει δεδομένα.
5.Λογικά ναι αλλά θα μπορεί μόνο να στέλνει δεδομένα και όχι να δέχεται δεδομένα από το server.
6.Σίγουρα θα είναι status code 400 και λογικά θα είναι 422.
7.Σίγουρα θα είναι status code 500 και λογικά θα είναι 501.
8. Δεν γίνεται να κάνω login με get γιατί ο χρήστης θα πρέπει να στείλει κάποια δεδομένα.
9. Και οι δυο μέθοδοι θα δημιουργήσουν κάτι στον server. Ωστόσο, η πολλαπλή χρήση put θα έχει το ίδιο αποτέλεσμα 
   ενώ η πολλαπλή χρήση post θα δημιουργήσει πολλαπλά παράγωγα.
10.Θα χρησιμοποιήσουμε μέθοδο post ή put
